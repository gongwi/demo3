# my-planner

## brief introduction
```
这是一个单页面开发的时间管理网页。该项目分为两个部份，第一部份是每日待办事项（每个事项有单独的计时功能）和每周事项（意为习惯养成），第二部份是统计功能，对前面事项的完成情况及所用时长等进行实时统计，让用户明确地知道自己的时间都用到了哪里。
```
